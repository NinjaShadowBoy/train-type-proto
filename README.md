 PROJECT NAME: TRAIN TYPE
The train type project is one which involves developing a web platform allowing users to practice typing on a keyboard to improve their speed and accuracy. Users are able to choose typing exercises based on different levels of difficulties. During this exercise, a timer is displayed and the text to type will be represented line by line for the user. Finally, at the end of each exercise, statistics are shown like the typing speed.
